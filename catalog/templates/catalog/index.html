Hi! This documentation describes how to use this app using cURL which is a command line tool and library for transferring data with URLs. I will use the hard coded URLs and some predefined values for my own convenience.

1. Get a list of the existing products

To get a list of the existing products run the following command:

curl -X POST -H 'Content-Type: application/json' http://localhost:8000/product/list

2. Update a product price

To update a product price run the following command:

curl -X POST -H 'Content-Type: application/json' -d "{\"product_id\":1, \"price\":777.77}" http://localhost:8000/product/update

3. See the price history

To see the price history for a certain product run the following command:

curl -X POST -H 'Content-Type: application/json' -d "{\"product_id\":1}" http://localhost:8000/product/pricelog
